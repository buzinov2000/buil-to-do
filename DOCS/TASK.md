# Todo Bot — Telegram Bot

## Что строим

Простой личный Telegram-бот. Отправляешь список задач — бот возвращает красиво отформатированное сообщение с кнопками. Нажимаешь кнопку с номером — задача зачёркивается прямо в том же сообщении.

Никакого фронтенда. Один Python файл. Деплой на VPS.

---

## Флоу

1. Отправил боту список (каждая строка — задача):
   ```
   Сделать аниматик по сцене 3
   Прислать правки Диме
   Обновить портфолио
   ```

2. Бот отвечает одним сообщением:
   ```
   📋 Задачи на сегодня

   ◻️ 1. Сделать аниматик по сцене 3
   ◻️ 2. Прислать правки Диме
   ◻️ 3. Обновить портфолио

   [1] [2] [3]
   ```

3. Нажал `2` — сообщение редактируется:
   ```
   📋 Задачи на сегодня

   ◻️ 1. Сделать аниматик по сцене 3
   ✅ ~~2. Прислать правки Диме~~
   ◻️ 3. Обновить портфолио

   [1] [3]
   ```
   Выполненные кнопки исчезают из ряда.

4. Когда всё выполнено — бот редактирует сообщение, убирает кнопки, добавляет 🎉 Всё готово.

## Структура данных

Хранить в SQLite активные списки — на случай если бот перезапустился, состояние не теряется.

```
tasks (
  message_id   INTEGER,   -- id сообщения в Telegram
  chat_id      INTEGER,   -- id чата
  items        TEXT,      -- JSON: [{text, done}, ...]
  created_at   DATETIME
)
```

---

## Команды бота

- /start — приветствие и инструкция
- /clear — убрать кнопки с последнего активного сообщения (сброс)

Любое другое сообщение = новый список задач.

---

## Технические детали

- Форматирование: Telegram MarkdownV2
- Зачёркивание: ~текст~ в MarkdownV2
- Кнопки: InlineKeyboardMarkup, callback_data = индекс задачи + message_id
- При нажатии кнопки: editMessageText + editMessageReplyMarkup
- Выполненные задачи убираются из кнопок
